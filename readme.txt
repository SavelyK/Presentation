Добрый день. 
В сборке 3 папки.  В папке core содержаться описания сущностей в библиотеке PresentationDomain и бизнес логика относящаяся к сущностям в библиотеке PresentationApplication.
В папке infrastructure содержится библиотека PresentationPersistence в которой описаны сценарии работы с базой данных. 
В папке Presentation расположен сервис api. Библиотеки PresentationPersistence и PresentationApplication подключаются к сервису api в виде отдельных сервисов. Строка подключения к бд вынесена в appsettings.json строка подключения - "DbConnection". 
API подключен к сваггер песочнице – при запуске происходит ридирект в сваггер песочницу где можно посмотреть контроллеры и эндпоинты. Пока что в проекте один контроллер Presentation с двумя методами 
Post – для создания презентации и 
Get – для получения информации о презентации.
Пока думаю над тем, как правильно связать между собой посетителей и презентации.
